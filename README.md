# 🚀 Стартап-идея: Приложение для поиска единомышленников


# Шаг 1. Уточнение выбранного сегмента

Мы рассматриваем сегмент **молодых людей 18–30 лет (студенты и молодые специалисты) в крупных городах России**, которые хотят участвовать в совместных активностях (настольные игры, спорт, онлайн-игры), но часто не имеют компании.

Проблема проявляется **в свободное время — вечером, на выходных или во время планирования досуга**, когда человек хочет заняться активностью, но не может быстро найти людей с теми же интересами.

Этот сегмент выбран на старте, поскольку:
- проблема возникает **часто и регулярно** (досуг и развлечения);
- сегмент **легко доступен для интервью и тестирования** (университеты, соцсети, игровые сообщества).


# Шаг 2. Сбор исходных наблюдений (desk research)

1. Во многих **Telegram и Discord чатах** существуют каналы «ищу компанию для игры», что показывает постоянную потребность пользователей находить людей для совместных активностей.

2. В игровых сообществах (например, по настольным играм) часто встречаются посты **«кто хочет поиграть сегодня?»**, что говорит о проблеме поиска игроков в конкретный момент времени.

3. Многие пользователи используют **сообщества ВК и форумы** для поиска игроков, однако поиск там занимает много времени и требует ручного общения.

4. Популярность **антикафе и игровых клубов** показывает высокий спрос на офлайн-встречи и совместные активности.

5. Сервисы знакомств активно используются молодёжью, что показывает готовность людей **знакомиться через приложения**.

6. В онлайн-играх игроки часто ищут **команды или напарников через Discord**, так как встроенные механизмы поиска игроков часто неудобны.

7. В университетах студенты часто создают **локальные чаты для поиска компании**, что показывает наличие постоянной потребности.

8. В приложениях для событий люди могут найти мероприятие, но **не могут быстро найти компанию для участия**.


# Шаг 3. Паспорт сегмента

**Кто:**  
студенты и молодые специалисты 18–30 лет.

**Контекст:**  
свободное время (вечера, выходные), желание заняться активностью.

**Цель:**  
быстро найти компанию для совместного занятия.

**Триггер:**  
желание поиграть, заняться спортом или посетить мероприятие.

**Текущая замена:**  
поиск через друзей, Telegram-чаты, Discord, форумы.

**Потери:**  
- потеря времени на поиск  
- отказ от активности  
- социальная изоляция  

**Ограничения:**  
- безопасность встреч  
- ограниченный бюджет  
- необходимость достаточного количества пользователей  

**Критерии выбора:**

1. скорость поиска людей  
2. количество пользователей  
3. безопасность  
4. удобство приложения  
5. фильтрация по интересам  

**Каналы доступа:**

- университеты  
- Telegram-чаты  
- Discord-сообщества  
- социальные сети  
- игровые клубы и антикафе  


# Шаг 4. Persona

## Персона: Алексей, 21 год

Алексей — студент университета, живёт в большом городе. Он любит настольные игры и онлайн-игры, но его друзья не всегда свободны. Иногда он хочет быстро найти компанию для игры или активности.

### Цели

- найти компанию для игр  
- проводить больше времени с людьми  
- участвовать в новых активностях  
- расширять круг знакомств  

### Боли / фрустрации

- друзья заняты или не заинтересованы  
- сложно найти людей с теми же интересами  
- поиск через чаты занимает много времени  
- многие мероприятия требуют готовой компании  

### Контекст жизни

- ограниченный бюджет  
- активное использование смартфона  
- много времени проводит онлайн  

### Текущий процесс

1. пишет друзьям  
2. ищет людей в чатах  
3. иногда отказывается от активности  

### Критерии выбора

1. скорость поиска  
2. количество пользователей  
3. безопасность  
4. простота приложения  
5. интересы пользователей  

### Возражения / риски

- недоверие к незнакомым людям  
- мало пользователей  
- небезопасные встречи  

### Где «обитает»

- Telegram  
- Discord  
- VK  
- игровые клубы  
- университетские сообщества

# Шаг 6. Карта инсайтов и гипотез

## Инсайты

1. Люди хотят заниматься активностями, но часто **не имеют компании**.
2. Поиск людей через чаты и форумы **занимает много времени**.
3. Пользователи готовы знакомиться через приложения.
4. Важно наличие **людей рядом**.
5. Доверие и безопасность являются важными факторами.
6. Большинство активностей происходит **в свободное время вечером и на выходных**.
7. Университеты и игровые сообщества — хорошие точки для привлечения пользователей.

## Гипотезы

1. Пользователи испытывают регулярную проблему поиска компании для активностей.  
2. Пользователи готовы использовать отдельное приложение для поиска людей по интересам.  
3. Геолокация и поиск людей рядом являются ключевой функцией.  
4. Пользователи готовы оставлять контакт для доступа к сервису.  
5. Пользователи готовы встречаться с новыми людьми для совместных активностей.  
6. Студенты — наиболее активный сегмент для старта продукта.  
7. Удобство и скорость поиска важнее дополнительных функций.

## ТОП-3 гипотезы

**Гипотеза 1**  
H: Для молодых людей 18–30 лет ключевая боль — поиск компании для активности.

Проверка: интервью с пользователями.  
Критерий: ≥70% подтверждают проблему.

**Гипотеза 2**  
H: Пользователи готовы использовать приложение для поиска людей по интересам.

Проверка: демонстрация прототипа.  
Критерий: ≥60% готовы попробовать сервис.

**Гипотеза 3**  
H: Функция поиска людей рядом является основной ценностью сервиса.

Проверка: опрос пользователей.  
Критерий: ≥50% называют её главной функцией.
